# NTUSTapp
一個屬於台科人的 app！

## 開發工具

### Timetable (In progress)
- 每週課表
- 教室查詢
- 離線瀏覽

### Score
- 學期成績
- GPA即時試算
- 修課歷史紀錄

### Other
- 搜尋空教室
