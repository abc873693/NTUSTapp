# NTUSTapp
> 一個屬於台科人的 App

## 開發工具
React Native ~~with Expo~~
- 於 v1.0.0 脫離 Expo 框架，直接採用 native code

## Features
### Timetable
- 個人課表
- 課程資訊 (In progress)
- 離線瀏覽

### Score
- 學期成績
- GPA即時試算
- 修課歷史紀錄

### Other
- 搜尋空教室
